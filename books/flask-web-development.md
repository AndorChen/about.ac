## 关于本书

[Flask](http://flask.pocoo.org/) 是使用 Python 开发的 Web 框架，用到了 Werkzeug 和 Jinja2 等包。Flask 是一个微型框架，但别被这个“微”迷惑了。其实 Flask 像其他框架一样强大，可以做很多事情，不管时小型应用还是大型应用都能胜任。

《Flask Web 开发》这本书全面介绍 Flask，并且手把手教你如何开发一个现代化的 Web 应用。
