作者

David McFarland 是 Web 开发者、老师和作者。他从 1995 年开始开发网站，那年设计了自己的第一个网站——一份写给通信专家看到的在线杂志。他曾任教于加州大学伯克利分校新闻研究生院、电子艺术中心、波特兰艺术学院和波特兰州立大学，教授 Web 设计。他现在是在线教育网站 Treehouse 的教学团队主管。
